# PPTX 페이지 번호 및 통합 작업 도구

우리은행 AX를 위한 AI Agent 구축 제안서 (III. 기술 부문) PPTX 파일들의
페이지 번호 삽입 및 단일 파일 통합 자동화 도구입니다.

## 폴더 구조

```
propsal/
│
├── .env                        # 환경 변수 설정 파일
├── requirements.txt            # Python 패키지 목록
├── runvenv.bat                 # 가상환경 활성화 CMD 실행
│
├── cmd/                        # 작업 실행 배치 스크립트
│   ├── pagenum.bat             # 페이지 번호 표시 작업 실행
│   └── pptmerge.bat            # PPTX 통합 작업 실행
│
├── src/                        # Python 소스 코드
│   ├── add_page_numbers.py     # 페이지 번호 삽입 스크립트
│   └── merge_pptx.py          # PPTX 파일 통합 스크립트
│
├── .venv/                      # Python 가상환경 (로컬 전용)
│
├── pptx/                       # 원본 PPTX 파일 (34개)
│   └── 우리은행 AX를 위한 AI Agent 구축 제안서_Ⅲ. 기술 부문_*.pptx
│
├── pptx_backup/                # 원본 백업 (34개, 최초 실행 시 자동 생성)
│   └── 우리은행 AX를 위한 AI Agent 구축 제안서_Ⅲ. 기술 부문_*.pptx
│
└── output/                     # 작업 결과 출력 폴더
    └── YYYYMMDD-HH24MISS/      # 실행 시각별 타임스탬프 폴더 (자동 생성)
        ├── 목차페이지정보.xlsx  # 파일별 페이지 범위 목차 (pagenum 실행 시)
        ├── 우리은행 ...*.pptx  # 페이지 번호 적용된 개별 파일 34개
        └── III.기술부문.pptx   # 전체 통합 파일 (pptmerge 실행 시)
```

## 환경 변수 (.env)

| 변수 | 기본값 | 설명 |
|------|--------|------|
| `PPTX_DIR` | `./pptx` | 원본 PPTX 파일 폴더 |
| `OUTPUT_DIR` | `./output` | 작업 결과 출력 루트 폴더 |
| `MERGE_OUTPUT` | `III.기술부문.pptx` | 통합 파일명 |
| `PAGE_NUM_FONT_SIZE` | `11` | 페이지 번호 폰트 크기 (pt) |
| `PAGE_NUM_COLOR` | `606060` | 페이지 번호 글자 색상 (RGB hex) |
| `PAGE_NUM_MARGIN_RIGHT` | `0.3` | 우측 여백 (inch) |
| `PAGE_NUM_MARGIN_BOTTOM` | `0.2` | 하단 여백 (inch) |
| `PAGE_NUM_WIDTH` | `1.0` | 페이지 번호 텍스트 박스 너비 (inch) |

## 실행 방법

### 1. 페이지 번호 삽입 (`cmd\pagenum.bat`)

```
cmd\pagenum.bat
```

- `pptx/` 폴더의 PPTX 파일을 파일명 순으로 정렬
- 각 파일의 슬라이드 마스터/레이아웃 우측 하단에 슬라이드 번호 placeholder 추가
- 파일 간 연속 페이지 번호 유지 (`firstSlideNum` 설정)
- 결과 파일을 `output/YYYYMMDD-HH24MISS/` 에 저장 (원본 유지)
- 목차 엑셀(`목차페이지정보.xlsx`)도 동일 폴더에 함께 생성

**목차페이지정보.xlsx 구성:**

| 파일명 | 슬라이드 수 | 시작페이지 | 끝페이지 |
|--------|------------|-----------|---------|
| ...1. 구성방안.pptx | 253 | 1 | 253 |
| ...2.1 개발 대상 업무.pptx | 6 | 254 | 259 |
| ... | | | |
| ...2.3 시스템 접근~5.pptx | 57 | 1005 | 1061 |

### 2. PPTX 통합 (`cmd\pptmerge.bat`)

```
cmd\pptmerge.bat
```

- `pptx/` 폴더의 PPTX 파일 34개를 하나의 파일로 병합
- 이미지, 차트, SmartArt, 임베딩 파일 포함 전체 리소스 보존
- 결과 파일을 `output/YYYYMMDD-HH24MISS/III.기술부문.pptx` 에 저장
- 총 1,061슬라이드 / 약 202 MB

### 3. 가상환경 직접 실행 (`runvenv.bat`)

```
runvenv.bat
```

- `.venv` 가상환경이 활성화된 CMD 창을 열어 수동으로 스크립트 실행 가능
